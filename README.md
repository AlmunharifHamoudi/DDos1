📦 Installation Steps (Kali Linux - with sudo)

```bash
sudo apt update && sudo apt upgrade -y
```

```bash
sudo apt install git -y
```

```bash
sudo apt install python3 -y
```

```bash
sudo apt install python3-pip -y
```

```bash
git clone https://github.com/AlmunharifHamoudi/DDos1.git
```

```bash
cd DDos1
```

```bash
pip install --break-system-packages requests httpx aiohttp colorama urllib3 certifi charset-normalizer idna pycryptodome
```

```bash
python3 DDos.py
```
